Student Information System (C++)
A console-based Student Information System built in C++ using a Singly Linked List as the underlying data structure. This project demonstrates fundamental CRUD (Create, Read, Update, Delete) operations — making it a great reference for anyone learning data structures and management system design.
Features

Add a new student record
Search a record by Roll Number
Modify an existing record
Delete a record

Tech Stack

Language: C++
Data Structure: Singly Linked List
Concepts: CRUD, Dynamic Memory Allocation, Pointers
